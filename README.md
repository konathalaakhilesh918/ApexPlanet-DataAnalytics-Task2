# ApexPlanet-DataAnalytics-Task2
Exploratory Data Analysis (EDA) &amp; Business Intelligence using Python (Pandas, Matplotlib, Seaborn)    
## 🔹 Overview

This project is part of the ApexPlanet Data Analytics Internship.

In this task, I performed Exploratory Data Analysis (EDA) on a sales dataset to extract meaningful insights and understand business patterns.

---

## 🔹 Dataset

* Contains 1000 records of customer transactions
* Includes customer details, product info, and sales data

---

## 🔹 Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 🔹 Key Steps Performed

* Data loading and preprocessing
* Converted `Order_Date` to datetime format
* Descriptive statistics using `.info()` and `.describe()`
* Data visualization using charts
* Business question analysis
* Correlation analysis using heatmap

---

## 🔹 Key Insights

* 📈 **Electronics category** generated the highest revenue (~50M)
* 💻 **Laptop and Mobile** are top-selling products
* 📅 Sales peak during **March and festive months**
* 👨 Male customers contribute slightly more revenue than female customers
* 💰 Over **50% of orders are high-value**

---

## 🔹 Visualizations

* Sales distribution (Histogram)
* Category-wise sales (Bar chart)
* Monthly sales trend
* Correlation heatmap

---

## 🔹 Conclusion

EDA helped uncover important trends in sales, customer behavior, and product performance, which can support data-driven business decisions.
